*파트너스 활동을 통해 일정액의 수수료를 제공받을 수 있음

얼마전 디스커스에서 또 한번의 패치를 하고 그동안 편법으로 하던 한글세팅의 방법이 변경되었다. 기존의 편법으로 이용하던 이용자들은 피해가 없지만, 신규로 블로그에 유입되는 사람들은 또 한번의 난항을 겪고있다.


디스커스(Disqus))

새로 변경된 내용

그동안에는 select box를 수정하여 쉽게 en을 ko로 변경하여 사용을 하였다.
그러나 이 방법은 이제 적용되지 않는다. 2020년 12월 기준으로는 selectBox 형식이 아닌 div로 UI를 그려내고 있어 이용자들이 쉽게 한글로 변경하기는 쉽지 않다. 거두절미하고 쉽게 설명하도록 하겠다.

Quick Start [이미지 새 창으로 보기]

disqus 전송구간 파라메터 변경
1. 개발자도구 (또는 F12)를 누른다.
2. 개발자도구의 Search부분에서 forums/update를 검색한다.
3. ctrl + F로 Oa.a.call(t && t.useInternal ? "internal/forums/update" : "forums/update" 를 검색한다.
4. n.props.selection.forum && (e = dm({ 부분을 찾아서 브레이크 포인트를 건다.
5. e 부분에 마우스를 클릭하여 en -> ko로 변경한다.
6. 브레이크포인트 Next를 눌러 변경된 값을 전송한다.

이렇게 하면 역시 한글로 디스커스를 이용할 수 가 있다.

끝으로

정상적인 방법은 아니지만, 동일하게 디스커스를 한글로 설장이 가능하다. 왜 한국어를 빼먹었는지 아직까지도 의문이다. 2018년 최초로 한글세팅을 하는 방법을 작성하였다. 많은 사람들이 내 글을 읽고 성공하고 블로그에 퍼가주시니 감사할 따름이다.

이 방법또한 최초로 작성하여 공유하는 방법이다.
많은분들에게 도움이 되고 출처를 남겨 퍼가셔도 무방하니 많은 사람들이 한글세팅에 성공하길 바란다.

감사의 글

GodSchool.kr 님 께서 댓글로 한글세팅이 변경되었다는 사실을 알려주셨습니다. 알려주셔서 감사합니다. 덕분에 다시 방법을 찾아 정정하고 블로그에 발행하게 되었습니다. 모두 한글세팅에 성공하시고 좋은 블로그를 만들어가길 바랍니다.
